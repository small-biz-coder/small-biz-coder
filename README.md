## Projects

### Lawn Care Estimation App
Full-stack · React · TypeScript · Node.js · Express

A customer-facing estimation tool where users draw their yard using the Google Maps polygon API to calculate square footage, step through a multi-stage quote flow, and automatically receive a branded PDF estimate** by email.

- Built a custom Node.js / Express email server — no third-party email SaaS, full control over SMTP config and deliverability
- Server-side PDF generation with PDFKit, dynamically populated from form data
- React + TypeScript frontend, bundled with Vite, deployed to Apache shared hosting
- Handles CORS, environment variable management, and SPF/DKIM/DMARC configuration end-to-end
